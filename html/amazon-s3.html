Amazon S3

Οποιεσδήποτε αλλαγές στο file system, διαρκούν στο dyno μέχρι να τερματιστεί ή να επανεκκινηθεί. Κάθε dyno εκκινεί με ένα καθαρό αντίγραφο του file system, από το πιο πρόσφατο deploy.

Η εφαρμογη θα χρειαστεί, να αποθηκεύσει κάπου φωτογραφίες. Για αυτό το λόγο, θα χρησιμοποίησουμε την υπηρεσία Amazon S3. Η υπηρεσία είνα δωρεάν για 1 χρόνο.

Δημιουργία λογαριασμού
  1. Πηγαίνουμε στην ιστοσελίδα του Amazon Web Services και πατάμε Create a Free Account
  2. Πληκτρολογούμε διεύθυνση email, κωδικό και το όνομα του λογαριασμού
  3. Συμπληρώνουμε κάποιες πληροφορορίες επικοινωνίας όπως όνομα, τηλέφωνο, πόλη κ.α.
  4. Χρειάζετει, να δώσετε πιστωτική κάρτα. Δεν θα χρεωθείτε, εκτός αν ξεπεράσετε τα όρια του επιπέδου Free Tier
  5. Επιβεβαιώνουμε τον αριθμό τηλεφώνου
  6. Επιλέγουμε το Basic Plan, το οποίο είναι δωρεάν
  7. Συμπληρώνουμε κάποια πεδία ακόμα, για την εφατομίκευση της εμπειρίας μας
Δημιουργια bucket
  1. Κάνουμε login στο λογαριασμό μας εδώ
  2. Πάνω αριστερά, πατάμε στην καρτέλα Services και επιλέγουμε το S3 κάτω από το Storage
  3. Πατάμε Create Bucket, δίνουμε όνομα springeshop-bucket και επιλέγουμε Region EU(Frankfurt)
  4. Στην καρτέλα Configure options, πατάμε Next
  5. Στην καρτέλα Set Permission, πατάμε Next
  6. Στην καρτέλα Review, πατάμε Create bucket.
Δημιουργια IAM χρήστη

Για να έχουμε πρόσβαση στο bucket και να προσθέτουμε και να ανακτούμε φωτογραφίες προγραμματιστικά, χειάζεται να δημιουργήσουμε ένα IAM χρήστη.

  1. Πάνω αριστερά, πατάμε στην καρτέλα Services και επιλέγουμε το IAM κάτω από το Security, Identity, & Compliance
  2. Στην αριστερή στήλη πατάμε Users
  3. Πάνω αριστερά πατάμε Add user
  4. Πληκτρολογούμε User name springeshop-user, επιλέγουμε την επιλογή Programmatic access και πατάμε Next
  5. Πατάμε Attach existing policies directly, πληκτρολογούμε s3 στο filter policies, επιλέγουμε το AmazonS3FullAccess και πατάμε Next
  6. Πατάμε Create user, για να δημιουργηθεί ο χρήστης. Μετά θα εμφανιστούν το access key id και το access key secret, τα οποία θα χρειαστούμε στην εφαρμογή
Bucket policy

Για να εμφανίζει η εφαρμογή τις εικόνες, χρειάζεται να δώσουμε read access στο bucket.

  1. Στο S3 Management Console, πατάμε στο bucket springeshop-bucket
  2. Πατάμε στην καρτέλα Permissions
  3. Πατάμε στην καρτέλα Bucket Policy, προσθέτουμε την ακόλουθη πολιτική και πατάμε Save
    {

Angular - Components my-footer, shop-services, navigation-bar, product

Θα δημιουργήσουμε τα components my-footer, shop-services, navigation-bar και product, τα οποία θα χρησιμοποιήσουμε, για τις υπόλοιπες σελίδες εφαρμογής.

my-footer component

Δημιουργούμε ένα φάκελο components μέσα στο src/app. Στο terminal του Visual Studio Code, πληκτρολογούμε την ακόλουθη εντολή, για να δημιουργήσουμε το component my-footer. 

                
                            ng generate component my-footer

Αφού δημιουργηθεί, σέρνουμε τον φάκελο my-footer, στο φάκελο components. Το Angular CLI δημιουργεί τέσσερα αρχεία μέσα στο φάκελο my-footer :

  • my-footer.component.css : είναι το css του component και αφορά μόνο το συγκεκριμένο
  • my-footer.component.html : είναι το template του component, δηλαδή τι HTML πρέπει, να εμφανίσει η Angular, όταν χρησιμοποιούμε αυτό το component
  • my-footer.component.spec.ts : μας επιτρέπει, να κάνουμε testing στο component
  • my-footer.component.ts : περιέχει τα δεδομένα και τη συμπεριφορά του component

Παρακάτω, δίνεται το my-footer.component.ts και τα αρχεία του φακέλου my-footer.

my-footer.component.html : Το αρχείο html περιέχει ουσιαστικά 3 στήλες με συνδέσμους, που έχουν χρήσιμες πληροφορίες για το e-shop.q

Το αρχείο css αφορά μόνο το συγκεκριμένο component. Όταν υπάρχει δηλαδή ένας κανόνας για τα h1{ color : red}, αυτός εφαρμόζεται μόνο στο συγκεκριμένο component και όχι στην υπόλοιπη εφαρμογή.

product component

Δημιουργούμε το component product και το τοποθετούμε στο φάκελο components. Εμφανίζει ένα συγκεκριμένο προϊόν και στοιχεία του, όπως εταιρία, όνομα και τιμή. 

                
                            ng generate component product

Δημιουργούμε μια κλάση Product, η οποία θα αναπαριστά το προϊόν. Προσθέτουμε τα properties, που έχει το entity Product.java. Επίσης, δημιουργούμε τα αντίστοιχα models Category και Brand., που αναπαριστούν την εταιρία και κατηγορία του προϊόντος και περιέχονται στην κλάση Product.ts .
